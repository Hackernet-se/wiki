Tree är ett program som listar directories rekursivt.

**Installation**

`sudo apt-get install tree`

Lista directories rekursivt

`tree`

Skapa sig en överblick över ett programs filstruktur

`tree -h /etc/apache2/`

Hitta stora filer

`tree -h /var/ | grep G]`
